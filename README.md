# Podvozek

Před začatkem prace:
- nastudovat robotiku, elektroniku, 3D modelování
- udělat výzkum světu robotu
- promyslet potřebné cíle a způsoby jejich realizace

Na podvozku se musí umístit:
- 2 motory
- ultrasonický senzor
- baterii
- controller (Mictobit nebo Raspberry Pi)
- kabely

3D model:
- 2 kola
- modularní
